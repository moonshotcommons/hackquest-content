### 项目简介

***Counter*** 是一个简单而实用的计数器项目，它能够记录和更新数值。我们在前面的章节中使用 Rust 实现了一个原生的 Solana 计数器，本节我们将会结合之前学过的 Anchor 框架的知识，实现一个 Anchor 版本的计数器。  

通过这个简单项目的实现，我们将开始初步探索如何在 solana 上进行开发！🚀
