# Content/super

在上一节中，我们成功地定义了一个功能模块，这是我们 ***Counter*** 程序的关键组成部分。

现在，我们面临一个新的挑战：如何在这个新模块中直接使用我们之前定义的结构体和其他元素？

**Super 关键字**

```rust
super
```

在Rust中，`super` 关键字代表当前模块的父模块。因此，我们可以使用它来从父模块导入所有可用的项。这包括之前定义的结构体、枚举、常量、类型别名等。

**Syntax**

super

- 提示
    
    ```rust
    pub mod demo {
        use super::*;
    }
    ```
