# Content/权限控制

在获取到猫爸猫妈信息后，我们需要做一些控制，因为不能让两只小猫可以随意孕育小猫，不然系统不就乱套了吗！

在示例代码中，我们决定只有当猫妈和猫爸的主人都是函数的调用者时，才允许孕育新的小猫。

> 根据功能的需要，可以修改这里的权限控制条件。
> 

为了获取到猫爸猫妈的主人信息，我们直接通过ERC721的***ownerOf***函数来查询，只需要将TokenId作为入参传入即可。

**Syntax**

require

- 提示
    
    ```solidity
    require(owner == msg.sender, "error");
    ```
